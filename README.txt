This project implements a Markov-Chain groupme bot using python
It utilizes the following topics from class:
  1) The project can be easily deployed using Docker
  2) The applications uses a simple (one route) RESTfull API
  3) The project uses the cloud as a platform for machine learning

The docker file manages installation of the required platforms and libraries.
The core of the project is in the file server.py, which manages the REST api, and makes calls to the machin learning library
The final project was deployed on Azure to be demoed
